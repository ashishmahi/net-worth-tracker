### Brand Mark Color Contract

The sidebar brand icon container uses a CSS gradient — theme-adaptive, not hardcoded:

```
background: linear-gradient(135deg, var(--accent), oklch(0.55 0.18 270))
color: var(--accent-fg)
```

The SVG icon uses `fill="currentColor"` — inherits `var(--accent-fg)` from container. This means the icon automatically adapts to all three themes (Ledger, Vault, Studio) and both light/dark modes.

### Static Asset Colors (manifest.json, meta tags only)

| PWA `background_color` (light) | `#f7f4ed` |
| PWA `theme_color` | `#3d3480` |

Accent reserved for: sidebar brand mark container background, active nav indicator dot ring only.
