## v1.6 — Encrypted Export (Phases 19–21)

**Goal:** Optional passphrase protection for exported wealth data and safe import — starting with Web Crypto envelope encryption (Phases 19–20), then **password-protected zip** export/import with modal UX (Phase 21).

| Phase | Name | Goal (summary) |
|-------|------|----------------|
| **19** | Crypto Utilities | `cryptoUtils.ts` — AES-256-GCM + PBKDF2; `EncryptedEnvelope`; typed wrong-passphrase errors; Vitest |
| **20** | Settings UI — Encrypted Export & Import | Inline passphrase export/import, envelope auto-detect, inline errors |
| **21** | Passphrase modals + zip | `@zip.js/zip.js` (`wealthDataZip`), AlertDialog export/import, `.zip` only import; legacy JSON picker path removed |
