3. **New component `src/components/NetWorthOverTimeCard.tsx`:**
   - **Props:** `history: import('@/types/data').NetWorthPoint[]`, **`recordBlockedMessage: string | null`** (pass through from Dashboard for optional insufficient-state helper line per **11-UI-SPEC** when recording is blocked).
   - **Sort:** `const sorted = [...history].sort((a, b) => new Date(a.recordedAt).getTime() - new Date(b.recordedAt).getTime())`.
   - **Branch:** If **`sorted.length < 2`**: render a **`Card`** with **`text-muted-foreground`** secondary text; **`CardHeader`** / title area must include the **exact** heading string **`Need two snapshots to see a trend`**. Body explains **at least two snapshots over time** and points users to **Record snapshot** / **Record another snapshot** per **11-UI-SPEC** copy table; if **`recordBlockedMessage`** is non-null, show **one** short optional line mirroring the Dashboard tone (do not duplicate long Settings text).
   - **If `sorted.length >= 2`:** render a **`Card`** with section title **`Net worth over time`** (**`text-lg`**, semibold per UI-SPEC). Inside, use **`ChartContainer`** from **`@/components/ui/chart`** with a **`config`** object mapping one series (e.g. **`totalInr`**) to **`hsl(var(--chart-1))`** — **no** raw hex. Wrap **Recharts** **`LineChart`** + **`CartesianGrid`** (muted **`--border`**) + **`XAxis`** + **`YAxis`** with **compact** tick formatters: **Y** = INR (**`en-IN`**, `style: 'currency'`, `currency: 'INR'`, **`maximumFractionDigits: 0`** to match dashboard net worth style unless UI looks cramped). **`XAxis`:** minimize clutter (e.g. **few ticks**, short date format derived from `recordedAt` ISO strings — implementation picks tick count from span). Use **`ChartTooltip`** + **`ChartTooltipContent`** so tooltip stays **compact** (amount-forward + short date, **one** primary line per **D-04**). Series: **`Line`** with **`type="monotone"`**, stroke from chart token; set **`isAnimationActive={false}`**. Optional **very light** **`Area`** fill under the line only if it stays subtle and uses the same token at low opacity — else **line only**.
   - **Responsive height:** wrapper **`w-full`** + fixed Tailwind height in **240–320px** range (e.g. **`h-[280px]`** or **`min-h-[240px]`**) per **11-UI-SPEC**.
