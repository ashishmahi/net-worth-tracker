    1. Add a **`Dockerfile`** at repo root with **two stages**:

    **Stage `build` (name with `AS build`):**
    - **`FROM node:20-alpine`** (or `node:22-alpine` if you standardize on 22 — pick one LTS and use it consistently).
    - **`WORKDIR /app`**
    - **`COPY package.json package-lock.json ./`**
    - **`RUN npm ci`**
    - **`COPY . .`**
    - **`RUN npm run build`** (must run `tsc -b && vite build` per package.json)

    **Final stage:**
    - **`FROM nginx:alpine`**
    - **`COPY --from=build /app/dist /usr/share/nginx/html`**
    - **`COPY docker/default.conf /etc/nginx/conf.d/default.conf`**
    - **`EXPOSE 80`**
    - No `CMD` override required — nginx:alpine default is fine.

    2. Do **not** add `ARG` for Vite `base` in this phase (Phase 24) unless CONTEXT is updated — default build uses `/`.

  </action>
