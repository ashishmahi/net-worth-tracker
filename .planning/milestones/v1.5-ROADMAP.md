## v1.5 — Debt & Liabilities (Phases 14–18)

**Goal:** Full liabilities layer — `DataSchema` migration, pure calculation helpers, property enrichment, standalone **Liabilities** CRUD page, and dashboard integration (debt-adjusted net worth, Total Debt row, debt-to-asset ratio).

| Phase | Name | Goal (summary) |
|-------|------|------------------|
| **14** | Schema & Migration | `liabilities` + `LiabilityItemSchema`, migration, tests; negative snapshot totals |
| **15** | Calculation Utilities | `liabilityCalcs.ts` — sum standalone, sum all debt, net worth, ratio |
| **16** | Property liability enrichment | Optional lender + EMI on property; hint to Liabilities page |
| **17** | Liabilities page CRUD | Standalone loans CRUD, sidebar nav, badges, empty state |
| **18** | Dashboard & net worth | Headline + snapshots use `calcNetWorth`; Total Debt row; ratio |
