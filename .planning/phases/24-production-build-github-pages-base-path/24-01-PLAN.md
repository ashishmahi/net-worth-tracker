  <action>
    1. Change the default export to **`defineConfig(({ mode }) => { ... })`** (callback form).

    2. Import **`loadEnv`** from **`vite`** alongside **`defineConfig`**.

    3. Inside the callback, call **`loadEnv(mode, process.cwd(), '')`** and read **`env.BASE_URL`**.

    4. Implement **`normalizeBaseUrl(raw: string | undefined): string`**:
       - If **`raw`** is **`undefined`**, **`null`**, or **only whitespace**, return **`'/'`**.
       - Trim **`raw`**. If empty after trim, return **`'/'`**.
       - Ensure a **leading** slash: if the string does not start with **`/`**, prepend **`/`**.
       - Ensure a **trailing** slash for non-root: if the result is **`'/'`**, return **`'/'`**; otherwise if it does not end with **`/`**, append **`/`**.

    5. Set **`base: normalizeBaseUrl(env.BASE_URL)`** in the returned config object.

    6. Preserve existing **`plugins`** and **`resolve.alias`** (`**@**` → **`path.resolve(__dirname, './src')`**) unchanged except for moving them inside the returned object.

    7. Do **not** add **`import.meta.env`** rewrites for `base` in application code — not required for this app (no react-router **`basename`**).

  </action>
